=======================
PETSHOP
=======================

Este site foi desenvolvido para a fase 2 da cadeira de Fundamentos de Sistema Web com o objetivo de apresentar os conceitos e funções nela estudadas. Tem como funcionalidades, cadastro de clientes e pet, login, envio de formulários, agendamento.

=======================
Instalação
=======================

Abra o arquivo index.html utilizando o navegador.

=======================
Uso
=======================

O site é composto por abas de navegação na parte superior, entre elas você irá navegar pelas páginas disponíveis.
